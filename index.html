GreenCap K8s

Gestão Simples de Clusters Kubernetes

Plataforma web para gerenciar os clusters Kubernetes que você já tem — workloads, redes, armazenamento, escalonamento e mais, com permissões por usuário. Uma alternativa leve a ferramentas como OpenShift e Rancher, ideal para estudo, desenvolvimento e testes em pequenas e médias equipes.

O Problema

Kubernetes ficou poderoso — e distante de quem só quer estudar, testar e construir

🧩

Complexidade excessiva

OpenShift e Rancher trazem camadas pensadas para operações enterprise, não para times pequenos ou estudantes.

🏗️

Infraestrutura pesada

Exigem componentes adicionais só para funcionar — custo operacional alto para clusters de estudo, dev e teste.

📈

Curva de aprendizado alta

Ferramentas construídas para especialistas, não para iniciantes.

A Solução

Uma UI leve e plug-and-play sobre o Kubernetes que você já opera

O que é

  • ✓ Monólito Spring Boot dockerizado — plug-and-play
  • ✓ Gerenciador ativo de clusters existentes (não provisiona clusters novos)
  • ✓ Executa ações reais: criar, escalar, reiniciar, suspender, disparar
  • ✓ Autorização 100% via RBAC nativo do Kubernetes

O que não é

  • ✕ Não provisiona clusters — você traz o seu (minikube, kind, gerenciado, on-prem)
  • ✕ Não é somente leitura
  • ✕ Não duplica controle de acesso próprio
  • ✕ Não é uma plataforma enterprise pesada

Registre seu cluster

Conheça a Plataforma

Funcionalidades

Tudo que você precisa para operar clusters no dia a dia

Global — visão de infraestrutura

  • Múltiplos Clusters e Namespaces
  • Nodes, PersistentVolumes e StorageClasses
  • Registry interno de imagens

Project — por Namespace

  • Topologia visual interativa
  • Observability: Dashboard, Events, Metrics
  • Workloads: Pods, Deployments, StatefulSets, Jobs, CronJobs
  • Networking, Parameters, Auto Scaling, Storage, Helm

4 formas de deploy

  • Deploy Application — a partir de imagem
  • Import Compose — traduz docker-compose.yml automaticamente
  • Deploy from Dockerfile — build via Kaniko + deploy em um fluxo
  • Deploy from Helm — wizard guiado

Developer Experience

  • Templates Catalog — apps completas prontas para deploy em 1 clique (ex.: Flask + PostgreSQL, MongoDB cache-aside)
  • Kubernetes Operators via OLM — catálogo, instalação e status em tempo real
